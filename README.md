This is a frontend application for managing employee data. The application allows users to add, delete, search and edit employee information.

Tech Stack:
React
Redux Toolkit
Material-UI
React-hook-form
Axios

Project setup:

1. Clone the repository:
   git clone https://github.com/tanayanene/CRUD-Application.git
   
3. Install dependencies
   npm i
   
5. Run the application
   npm run dev
